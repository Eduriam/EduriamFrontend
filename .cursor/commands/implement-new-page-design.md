# Implement new page design

Based on the provided links to pages and Figma prototype pages, implement the new page design. DO NOT change the functionality of the page. Change only the design.

## Input

- Figma prototype pages and components
- Links to the pages in the codebase

## Workflow

1. ANALYZE all the provided Figma prototype pages and components.
2. IMPLEMENT the new design of the pages.
3. REFACTOR the code so that it follows the implementation guidelines.
4. RUN `yarn build` and `yarn lint` to verify that the feature is working. Fix any issues.

## Details

- All implementation should be responsive!
- Always follow the implementation guidelines defined in `.cursor/rules/implementation-guidelines.mdc`
- If you have any questions regarding the implementation you can ask.
- If the Figma page contains SVG images, download them and put them in a `public/images` directory
